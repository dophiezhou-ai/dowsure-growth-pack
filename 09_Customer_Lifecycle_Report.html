3W1H：销售先回答四个问题

问题销售判断输出
Where 客户在哪Amazon PR、存量客户、服务商、同行 Referral、卖家社群、活动线索。渠道优先级、首触 SLA、线索来源质量。
Who 客户什么样Amazon US 为主，月 GMV $10K-$300K，老板/运营能直接决策，有回款、备货、广告、账期或风险痛点。L1-L4 分层，确认是否 SA 主战场。
What 我们卖什么卖 AI Agent 服务和订阅，不卖金融。Trio 是 Agent 诊断后的执行动作。推荐 Agent、订阅级别、Trio 场景。
How 如何卖一个痛点切入，一个 Agent 诊断，一个低摩擦下一步。Free Check、Trial、Working Capital Map、转 KA。

阶段用户动作Agent 输出销售动作目标
Free Check填写店铺、GMV、回款、库存、广告、账期。经营压力评分、推荐入口 Agent。24h 内首触，确认层级和主痛点。MQL → SQL
First Value查看第一份诊断，判断是否准确。回款、备货、广告或账期的一条明确建议。只解释一个痛点，推动低摩擦下一步。SQL → Trial
Weekly Ritual每周查看现金流、库存、广告、风险提醒。本周该停什么、补什么、收什么、避什么风险。CRM 提醒回访，推动客户采纳建议。形成使用习惯
Subscribe把 Agent 输出纳入老板决策或经营例会。月度 ROI Proof、多 Agent 监控。推进 Starter / Pro / Scale。Paid Subscription
Trio Action在正确资金节点申请或预约评估。FastPay / TermPay / MoreFund 适配建议。做准入评估，不承诺额度、费率、审批。Trio 转化
Renew / Refer复盘价值，推荐同行。续费风险、增购机会、Referral 时机。推动续费、增购、转介绍和案例沉淀。NRR / Referral

Trio 嵌入场景：不是开场主卖点

Trio 动作最合适场景触发 Agent客户价值销售口径
FastPay店铺应收账款回流慢，Amazon 回款未到账但近期要补货或投广告。Capital + Sentinel加快资金流，减少等待回款造成的机会损失。先判断应收回流是否卡住经营，再判断 FastPay 是否适合。
TermPay下游账期过长，供应商付款与销售回款之间存在明显时间差。Deal + Capital + Sentinel优化账期安排，降低短期周转和人工协调成本。不是单纯延长账期，而是匹配采购、销售和回款节奏。
MoreFund旺季备货、爆品加单、广告放量、库存扩张。Insight + Spend + Capital抓住增长窗口，提高备货效率和利润弹性。先证明增长机会和库存需求，再判断是否需要 MoreFund。

销售与 CSM 的日常动作

每日

销售看 Pipeline

新增 SQL ≥3 个，跟进 Pipeline ≥5 个，客户回访 ≥2 次，CRM 100% 更新。

每周

推动 Weekly Brief

让 Trial / 订阅客户每周至少查看 1 次 Agent Brief，并采纳 1 条建议。

每月

复盘 ROI Proof

复盘成本降低、资金流加快、利润改善、时间/人员成本节省，推动续费和增购。

指标定义目标口径
Free Check Completion完成 3 分钟测算的线索数。作为 MQL → SQL 的前置信号。
