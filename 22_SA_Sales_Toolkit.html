01 · Lead Scorecard

SME 线索评分卡：先判断值不值得跟

总分 100 分，70 分以上进入优先跟进，50-69 分进入培育，50 分以下不进入本周重点 Pipeline。

客户身份
20

真实跨境电商卖家，有平台店铺、订单、SKU、经营动作。

经营压力
30

增长、广告、库存、回款、供应商付款、账期或风险压力明确。

决策效率
20

老板、运营负责人或财务负责人可触达，能约下一步。

可验证价值
30

能进入轻诊断或 Trial，14 天内有机会看到 Agent 输出价值。

分数段客户状态销售动作经理检查
85-100痛点强、决策快、可快速试用。当天约 Demo / Trial，48 小时内锁定下一步。是否进入本周重点 Pipeline。
70-84痛点明确，但数据或决策链需要补齐。先做 15 分钟 Discovery，再安排轻诊断。是否有明确阻塞项。
50-69符合画像，但痛点不够急或决策弱。进入培育池，推内容、案例和活动。是否有重新激活日期。
<50非主战场或本阶段价值难验证。不占用本周销售资源，可转市场长期教育。是否误放进 Pipeline。
02 · Sales SOP

从首触到付费：一线 SA 的标准动作

阶段目标必做动作退出条件
MQL 首触确认客户是否值得进入销售跟进。24 小时内触达，确认平台、类目、GMV 区间、当前最痛问题。有明确痛点和下一步，进入 SQL。
SQL 问诊找到客户最愿意解决的经营问题。围绕广告、库存、回款、供应商付款、账期、多平台管理问诊。明确推荐 Agent 和低摩擦下一步。
Free Check让客户先看到诊断价值。输出 1 页经营诊断：主要问题、影响、建议试用 Agent。客户愿意开 Trial 或约 Demo。
Demo / Trial证明 Agent 不是概念。展示客户场景下的建议，14 天内至少复盘 1 次。客户认可价值并进入付费讨论。
付费订阅成交 Agent OS 订阅。确认订阅级别、使用人、首月目标、成功标准。合同/订单完成，进入客户成功。
经营资金场景评估在 Agent 输出后评估回款、备货、账期服务。只基于客户明确场景和资料进入评估，不做承诺。转风控/财务流程或回到订阅服务。
03 · Discovery Sheet

问诊表：把客户痛点问准

基础经营问题

1. 目前主要平台和主营类目是什么？

2. 月 GMV 大概在哪个区间？最近 3 个月是增长、持平还是下滑？

3. 现在最想解决的是增长、广告、库存、回款、采购付款还是风险？

4. 这个问题如果 30 天不解决，会影响什么？

Agent 推荐问题

1. 有没有 SKU 不知道该加投还是停掉？

2. 广告花费增长后，利润有没有同步增长？

3. 平台回款慢几天，会不会影响补货或广告？

4. 供应商付款最卡的是订金、尾款还是账期？

04 · Agent Recommendation Rules

Agent 推荐规则：按痛点推荐，不按产品硬推

客户说法销售判断推荐 Agent下一步
“销量有，但不知道怎么继续增长。”增长决策卡住。Insight Agent做 SKU / 类目增长机会诊断。
“广告越花越多，利润没起来。”费用效率和库存节奏可能错配。Spend Agent做广告和费用效率诊断。
“平台回款太慢，补货跟不上。”现金流时间差影响经营。Capital Agent做现金流时间表，再判断回款提速或备货支持评估。
“供应商要先款，账期短。”采购付款节奏拖累现金流。Deal Agent做采购付款节奏诊断，再判断账期服务评估。
“多平台、多账户，看不过来。”资金和经营信息分散。Treasury Agent做资金和账户效率看板。
“店铺最近有异常/资料总是不齐。”风险和准入边界不清。Sentinel Agent做风险预警和资料准备。
05 · Trial Success Criteria

14 天 Trial 成功标准：必须看到价值，才推进付费

第 1-2 天

完成诊断目标

明确客户试用目标：找增长机会、降广告浪费、看现金流压力、看供应商付款节奏。

第 3-7 天

产出 Agent 建议

至少 3 条可执行建议，必须能落到 SKU、广告、库存、回款、采购或风险动作。

第 8-14 天

复盘并推进付费

完成 1 次价值复盘，确认订阅级别、首月目标和是否进入经营资金场景评估。

07 · Weekly Manager Checklist

销售经理周会检查表

勾选检查项达标标准未达标处理
☐新增 SQL每人每天 ≥3 个，且痛点和决策人完整。回看线索来源和首触质量。
☐Pipeline 跟进每人每天 ≥5 个，有下一步或阻塞原因。经理协助推进或淘汰。

☐合规抽查无额度、费率、审批、到账、增长承诺。立即纠正并再培训。
