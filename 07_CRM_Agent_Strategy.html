  • RevOps Console 全面接管 Pipeline 视图

  • Agent CRM 横向扩展至 CSM / 客成 / Risk Ops

关键风险与缓解

风险 1 · AE 抗拒变化
AE 习惯 Salesforce → 缓解：Phase 1 不强制切换，仅提供"Agent Sidekick"；用激励 + Top Performer 案例驱动迁移。
风险 2 · 数据迁移与合规
SF 5 年历史数据 + GDPR/PDPA 要求 → 缓解：SF 在 P1/P2 仍为系统记录主库；Memory 层只读访问；通过 HSBC 数据合规审计。
风险 3 · 模型幻觉影响成单
Agent 报价或合同生成幻觉 → 缓解：Sentinel 守卫 + 报价金额 ≥ $50K 必走人工双签；建立反馈环回训练。
风险 4 · 三层模型成本失控
Frontier API 调用费暴涨 → 缓解：缓存 + 路由策略（70% 走自托管 / 25% 域专 / 5% Frontier）；月度 cost-per-deal ≤ $35。
