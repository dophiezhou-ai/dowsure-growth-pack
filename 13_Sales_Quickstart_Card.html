6 大 Agent 一句话

Agent销售说法适合谁
Insight Agent帮你找下一个 SKU / 类目 / 关键词增长机会。增长卡住、选品和广告靠经验的卖家。
Deal Agent帮你看采购、供应商账期和付款节奏。供应商要先付款、备货压力大的卖家。
Spend Agent帮你判断广告、库存、物流的钱花得值不值。广告加预算、库存压货、ROI 不清的卖家。
Capital Agent帮你判断什么时候需要资金动作，以及该不该用 Trio。回款慢、补货急、现金流错配的卖家。
Treasury Agent帮复杂客户看多币种、FX 和资金调度。多站点、多币种、CFO 参与客户，SA 识别后转交。
Sentinel Agent帮你提前看店铺、准入和风控风险。怕被拒、怕麻烦、怕风险不清的卖家。

Trio什么时候嵌入对应 Agent客户价值

MoreFund旺季备货、爆品加单、广告放量、库存扩张。Insight + Spend + Capital抓住增长窗口，提升备货效率和利润弹性。

拉体验到复购的 6 步

步骤销售要做什么客户看到什么目标
1. Free Check让客户做 3 分钟应收回流 / 经营压力测算。一个清楚的痛点评分。MQL → SQL
2. First Value只解释一个最痛问题，不讲全套。回款、备货、广告或账期的一条明确建议。进入 Trial / 诊断
3. Weekly Ritual让客户每周看一次 Agent Brief。本周该停什么、补什么、收什么、避什么风险。形成使用习惯
4. Subscribe用价值证明推动 Starter / Pro / Scale。持续监控和月度 ROI Proof。付费订阅
5. Trio Action在正确场景后置 FastPay / TermPay / MoreFund。资金动作建议和合规边界。Trio 转化
6. Renew / Refer复盘价值，请求续费、增购和转介绍。成本降低、资金流加快、利润改善。复购 / Referral
