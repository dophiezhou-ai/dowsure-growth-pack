SA 销售话术与作战手册

这份不是市场材料，而是 SA 销售团队每天围攻 SME 中小 Amazon 卖家的 Sales Playbook：用 3W1H 快速判断客户、用低摩擦动作推进 Capital Agent 测算 / Growth Trial / AI Agent 订阅。

1. 统一销售口径

我们卖什么 Positioning

SA 团队卖的是 AI Agent 服务，不是金融产品。Trio 是 Agent 服务里的执行动作，不是销售开场主菜。

不要怎么卖 Redline

不要先卖金融、不要先谈费率、不要承诺额度和放款。先问客户的 Amazon 回款、备货、广告、供应商账期、FX 和风险。

销售成功定义 North Star

每天新增 SQL、推进 Pipeline、完成回访、开通测算/Trial、转付费订阅、请求 Referral。

1A. 6 大 Agent 的销售点

Agent工作方向SA 要卖的点Trio 关系
Insight AgentSKU、类目、关键词、竞品和增长机会。卖“下个增长点在哪里”。不直接卖 Trio；需要备货资金时转 Capital。
Deal Agent采购、供应商账期、订单节奏和付款安排。卖“采购和账期怎么更顺”。账期压力明确后，再判断 TermPay / MoreFund。
Spend Agent广告、库存、物流、平台费用和现金消耗。卖“钱花在哪里浪费了”。现金流被支出压住时转 Capital。
Capital Agent回款、备货、采购、广告之间的资金错配。卖“什么时候该用哪种资金动作，什么时候不该用”。FastPay / TermPay / MoreFund 是这里的执行动作。
Treasury Agent多币种、FX、账户余额和资金调度。SA 识别后转 KA/CGO，不强行成交。复杂资金动作由 KA/CGO 处理。
Sentinel Agent店铺健康、平台政策、账户异常和准入风险。卖“风险提前发现，适不适合先说清楚”。所有 Trio 动作前先过风险边界。

1C. Trio 三大金融产品嵌入逻辑

Trio嵌入场景对应 Agent销售价值话术方向
FastPay店铺应收账款回流慢，Amazon 回款未到账但近期要补货或投广告。Capital + Sentinel加快资金流，减少等待回款的机会损失。先判断应收回流是否卡住经营，再判断 FastPay 是否适合。
TermPay下游账期过长，供应商付款与销售回款之间存在明显时间差。Deal + Capital + Sentinel优化账期安排，降低短期周转和人工协调成本。不是单纯延长账期，而是匹配采购、销售和回款节奏。
MoreFund旺季备货、爆品加单、广告放量或库存扩张。Insight + Spend + Capital抓住增长窗口，提高备货效率和利润弹性。先证明增长机会和库存需求，再判断是否需要 MoreFund。

5. SME 可复制话术

转交型 · L1 / 复杂 L2

这个客户已经超出 SA 日常成交半径：多站点、CFO 参与、银行协同复杂。我们先完成识别和信息收集，然后转交 KA/CGO。SA 不在这个客户上消耗日常 Pipeline 产能。

L2 上沿 SME · 工贸卖家

王总，你的问题不是单笔额度，而是采购、库存和 Amazon 回款之间有时间差。我们先画一张 Working Capital Map，看看是采购端卡、回款端卡，还是长期周转卡。判断清楚后，再看 Capital Agent 服务内应该触发 TermPay、MoreFund 还是 FastPay。

L3 成长 SME · 核心客户

陈总，你们现在不是缺一个工具，而是广告、库存和回款节奏没有同步。建议先开 14 天 Growth Trial，让 Insight、Spend、Capital 三个 Agent 给你一份诊断。我们先看建议有没有用，再决定是否订阅，以及是否在 Capital Agent 服务内触发 FastPay / MoreFund。

L4 成熟小卖家 · 规模化客户

李老板，您先不用理解复杂系统。花 3 分钟填一下 Amazon 店铺、最近回款和备货情况，系统先判断是否适合 Capital Agent Lite 服务。FastPay 只是诊断后可能触发的执行动作；如果不适合，我们直接告诉您；如果适合，再讲材料、费用和下一步。

7. 常见异议处理

我不需要贷款

理解。我们也不是先卖贷款。Dowsure 先帮你做经营和资金诊断，判断回款、备货、广告、账期哪里错配。只有 Agent 诊断证明某个 Trio 执行动作有价值，才进入对应申请。

费率多少

费率一定要结合平台数据、回款周期、风险和使用场景评估。我们先测方案，再让你比较成本和收益，不会让你在没看清 ROI 的情况下使用。

AI Agent 靠不靠谱

Agent 不替客户做不可逆金融决策。它负责诊断、提醒和推荐；关键动作仍然需要客户确认，并以风控评估、银行审批和合同为准。

我已经有银行

这反而更适合。银行解决资金来源，Dowsure 解决资金如何贴近 Amazon 经营数据使用，提高额度使用效率、减少错配和人工判断成本。

11. 合规话术锁定

额度

不要承诺额度。标准口径：初步测算仅供参考，最终可用额度以完整材料、平台数据、资方规则和风控审核结果为准。

费率

不要承诺固定费率。标准口径：费用会根据产品类型、期限、风险评估和资方规则确定，评估后提供正式方案。

审批

不要承诺一定通过。标准口径：我们可以协助做准入评估和材料整理，审批结果以资方和风控审核为准。

到账

不要承诺固定到账时间。标准口径：我们会尽快推进流程，实际时间取决于资料完整度、审核进度和回款路径。
