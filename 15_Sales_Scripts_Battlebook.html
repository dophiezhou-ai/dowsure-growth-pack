销售话术与作战手册

这份不是市场材料，而是一线销售和销售管理者每天可用的 Sales Playbook：先分层，再诊断，再推进 Pipeline，最后完成 Agent OS 订阅、FastPay / TermPay / MoreFund attach 和 Referral。

1. 统一销售口径

我们卖什么 Positioning

Dowsure 是跨境电商卖家的 AI Growth & Capital OS。Agent OS 负责诊断经营和资金问题，Embedded Finance 负责把建议变成行动。

不要怎么卖 Redline

不要先卖贷款、不要先谈费率、不要承诺额度和放款。先问客户的 Amazon 回款、备货、广告、供应商账期、FX 和风险。

销售成功定义 North Star

SQL 质量、Pipeline 推进率、Paid Subscription、Finance Product Attach、Renewal、Expansion、Referral。

4. L1-L4 可复制话术

L1 · 顶级品牌

我们不建议先从单笔融资或费率开始聊。你们这个规模真正要看的是 cash conversion cycle、银行资金使用效率、FX 暴露和风险预警。Dowsure 可以先做一场 Capital Efficiency Meeting，把 Amazon 回款、供应商账期、银行额度和多币种资金放进一张 Capital Map，再判断是否值得做 limited-scope Pilot。

L2 · 工贸大卖

你们不是没有订单，而是订单增长后采购、库存、供应商付款和 Amazon 回款周期被拉长。我们先不急着推荐产品，先画一张 Working Capital Map，看哪一段最卡。如果卡在采购端，看 TermPay；卡在长期周转，看 MoreFund；卡在回款端，看 FastPay。

L3 · 成长品牌

你们现在最关键的是增长和现金流同步。Insight Agent 看增长机会，Spend Agent 看广告和库存支出，Capital Agent 判断 FastPay / MoreFund 是否适合。建议先开 14 天 Growth Trial，用真实数据看诊断是否能帮你做决策。

L4 · 中小卖家

您先不用理解复杂金融产品。花 3 分钟填一下 Amazon 店铺、回款和备货情况，系统先判断能不能加速回款、适不适合 FastPay。如果不适合，就当免费诊断；如果适合，我们再讲可用金额、费用和下一步材料。

6. 常见异议处理

我不需要贷款

理解。我们也不是先卖贷款。Dowsure 先帮你做经营和资金诊断，判断回款、备货、广告、账期哪里错配。只有诊断证明 FastPay、TermPay 或 MoreFund 有价值，才进入产品申请。

费率多少

费率一定要结合平台数据、回款周期、风险和使用场景评估。我们先测方案，再让你比较成本和收益，不会让你在没看清 ROI 的情况下使用。

AI Agent 靠不靠谱

Agent 不替客户做不可逆金融决策。它负责诊断、提醒和推荐；关键动作仍然需要客户确认，并以风控评估、银行审批和合同为准。

我已经有银行

这反而更适合。银行解决资金来源，Dowsure 解决资金如何贴近 Amazon 经营数据使用，提高额度使用效率、减少错配和人工判断成本。

7. Referral 固定动作

触发时点销售动作话术
Trial 有结果后请求推荐同阶段卖家。如果这次诊断对你有帮助，你身边是否有 2 位也在做 Amazon / Shopify 的朋友？我可以给他们同样做一次 Free 诊断。
首次使用 FastPay / TermPay / MoreFund 后请求供应商、同行、服务商 Referral。这套方式如果帮你解决了回款或账期问题，你供应链里应该也有类似卖家。你愿不愿意帮我们做 2 个高质量介绍？
L1 / L2 Pilot 成功后请求 Case、行业背书或生态引荐。如果 Pilot 结果达到预期，我们希望沉淀一个匿名 case，同时请你帮我们引荐 2-3 家业务周期类似的卖家。

8. CRM 必填字段

客户识别

客户分层、平台、GMV 区间、决策人、线索来源。

需求判断

主痛点、推荐 Agent、推荐产品、推荐订阅级别。

Pipeline

销售阶段、下一步动作、下次跟进时间、成交概率、Forecast 金额。

增长闭环

产品挂载潜力、增购潜力、Referral 潜力、流失风险。
