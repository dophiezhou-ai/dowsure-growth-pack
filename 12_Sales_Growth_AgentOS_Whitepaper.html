02 · Six Agents As Sellable Motions

先明确 6 大 Agent 的工作方向，再明确 SA 到底卖什么

SA 对外卖的是 AI Agent 服务，不是金融产品。Trio 三个产品是 Capital Agent / Treasury Agent 服务中可能触发的执行动作；销售开场必须先讲客户经营问题、Agent 诊断价值和订阅理由，再判断是否需要 FastPay / TermPay / MoreFund。

6 大 Agent工作方向SME 真实需求SA 销售点客户得到什么Trio 如何出现
Insight Agent
增长洞察		 看类目、SKU、关键词、竞品和销量机会。 不知道下一个增长点在哪里，广告和选品靠经验。 卖“下个增长机会在哪里”，不是卖报表。 14 天增长机会诊断、SKU / Category Opportunity Report。 通常不直接触发 Trio；当增长建议需要备货资金时，转入 Capital Agent。
Deal Agent
交易与供应商		 看采购计划、供应商账期、订单节奏和付款安排。 供应商要先付款，Amazon 回款晚，采购节奏被卡住。 卖“采购和账期怎么更顺”，不是单笔借款。 Supplier Term Map、采购付款节奏建议。 账期压力明确后，由 Capital Agent 判断是否需要 TermPay / MoreFund。
Spend Agent
支出效率		 看广告、库存、物流、平台费用和现金消耗。 钱花出去了，但不知道广告、库存、回款是否匹配。 卖“钱花在哪里浪费了，哪里应该停、哪里应该加”。 Spend Efficiency Report、广告与库存错配提醒。 当支出压力造成现金流缺口时，转入 Capital Agent 做资金动作判断。
Capital Agent
资金动作		 看回款、备货、采购、广告之间的 working capital 错配。 不是没有生意，而是现金流时间差影响补货和增长。 卖“什么时候该用哪种资金动作，什么时候不该用”。 Working Capital Map、资金动作建议、ROI Proof。 FastPay / TermPay / MoreFund 都是在这里被推荐、筛选和执行。
Treasury Agent
资金调度		 看多币种、FX、账户余额、回款路径和资金效率。 多站点、多币种、CFO 参与，已经超出普通 SA 成交半径。 SA 负责识别，不强行成交；符合条件转 KA/CGO。 FX / Treasury Optimization Memo、专项诊断入口。 复杂资金动作由 KA/CGO 评估，SA 不承诺额度、费率和审批。
Sentinel Agent
风险哨兵		 看店铺健康、平台政策、账户异常、风控准入和合规边界。 客户怕麻烦、怕被拒、怕影响店铺和资金安全。 卖“风险提前发现，适不适合先说清楚”。 Risk & Eligibility Monitor、准入预筛和风险提醒。 所有 Trio 动作前都必须经过风险边界判断。
SA 的标准定位：我们卖 AI Agent 服务，帮助 SME 卖家持续判断增长、支出、交易、资金和风险；Trio 不是开场主卖点，而是当 Agent 诊断证明有必要时才触发的执行动作。
02A · Sales Operating Method

不是所有客户都买 6 个 Agent：先单点成交，再升级多 Agent

6 Agent 是战略白皮书中的完整能力顺序，但 SA 不能要求所有 SME 一次性购买全套。销售作业方法是：先根据用户画像和行业痛点判断入口 Agent，再根据数据连接、使用频次、ROI Proof 和 Trio 场景逐步升级订阅级别。

02B · Trio Embedded Scenarios

Trio 三大产品嵌入场景

Trio 的角色是把 Agent 的经营诊断变成资金执行动作。目的不是让客户多借钱，而是降低客户资金成本、加快资金流、提升利润，并减少公司在回款、备货、账期、审批和人工判断上的时间成本与人员成本。

Trio 动作最合适场景触发 Agent客户价值销售口径
FastPay 店铺应收账款回流慢，Amazon 回款未到账但近期要补货或投广告。 Capital Agent + Sentinel Agent 加快应收回流，降低等待回款造成的机会损失。 “我们先看你的应收回流是否卡住经营，如果卡住，再判断 FastPay 是否适合。”
TermPay 下游账期过长，或供应商付款与销售回款之间存在明显时间差。 Deal Agent + Capital Agent + Sentinel Agent 优化账期安排，减少人工协调和短期周转压力。 “不是单纯延长账期，而是把采购、销售和回款节奏重新匹配。”
MoreFund 旺季备货、爆品加单、广告放量或库存扩张需要阶段性资金支持。 Insight Agent + Spend Agent + Capital Agent 抓住增长窗口，提高备货效率和利润弹性。 “先证明增长机会和库存需求，再判断是否需要 MoreFund 支持旺季备货。”
提成方向应与战略一致：基础提成看 Agent 订阅收入，增量提成看 Trio 执行动作和多 Agent 升级，续费和 Referral 作为长期奖励；所有提成必须以合规销售、真实使用和客户价值证明为前提。
03 · L1-L4 Commercial Packaging

SA 主攻 L3/L4，选择性处理 L2 上沿，L1 转交

L1

转交型客户

对象: 顶级品牌、多站点、多币种、CFO 参与。不是 SA 日常主战场。

  • 锚点案例: 林总 - 3C 配件 - 月 GMV $1.8M - Amazon US/EU/JP
  • SA 动作: 识别、记录、转交 KA/CGO
  • 不计入: SA 每日 SQL 目标
  • 原则: 不占用日常 Pipeline 产能
L2

L2 上沿 SME

对象: 月 GMV 接近 $300K、供应链重、有明确现金周期错配的卖家。

  • 锚点案例: 王总 - 家居类目 - 月 GMV $300K - Amazon US
  • 入口: Working Capital Map
  • 主推: Deal + Capital + Sentinel，按场景后置 TermPay / MoreFund
  • 成交: Scale subscription，Trio 作为服务内执行动作记录
L3

L3 成长 SME

对象: 月 GMV $50K-$150K、广告和库存投入增长快的 Amazon 卖家。

  • 锚点案例: 陈总 - Beauty 个护 - 月 GMV $90K - Amazon US + Shopify
  • 入口: 15 分钟增长资金诊断
  • 主推: Insight + Spend + Capital
  • 成交: Pro subscription；FastPay / MoreFund 由 Trial 结果后置判断
L4

L4 成熟小卖家

对象: 月 GMV $10K-$50K、老板直接决策、最关心回款和备货。

  • 锚点案例: 李老板 - 宠物配件 - 月 GMV $18K - Amazon US
  • 入口: 3 分钟应收回流测算
  • 主推: Capital Lite + Sentinel Lite
  • 成交: Starter / Pro subscription；FastPay 作为可能的执行动作
