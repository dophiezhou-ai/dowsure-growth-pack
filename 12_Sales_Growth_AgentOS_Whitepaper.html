02 · Six Agents As Sellable Motions

六个智能体如何被销售落地

每个 Agent 都必须有对应的客户痛点、买方角色、触发信号、销售动作、交付物和订阅理由。否则 Agent 只是概念, 不能成交。

Agent客户痛点买方角色销售切入点交付物订阅理由
Insight Agent
洞察官		 不知道下个增长品类、SKU 和关键词在哪里。 Founder、运营负责人、CMO “我们先给你一份 14 天增长机会诊断。” Category / SKU Opportunity Report 每周持续发现增长机会, 不是一次性报告。
Deal Agent
交易官		 供应商报价、账期和采购节奏难优化。 老板、采购、供应链负责人 “我们把供应商付款和账期空间画出来。” Supplier Term Map 持续优化采购、账期和供应商组合。
Spend Agent
支出官		 广告、库存、物流支出分散, ROI 不清。 运营负责人、广告负责人 “先找出广告和库存错配点。” Spend Efficiency Report 持续调整广告、库存、物流支出。
Capital Agent
资本官		 不知道什么时候该用 FastPay、TermPay、MoreFund。 老板、CFO、财务负责人 “不是借一笔钱, 是做资金组合。” Working Capital Recommendation 持续匹配资金工具, 形成金融产品 attach。
Treasury Agent
司库官		 多币种、FX、闲置资金和回款安排混乱。 CFO、财务、Treasury “把 FX 和闲置资金纳入同一张资金图。” FX / Treasury Optimization Memo 持续降低 FX 损耗、提高资金效率。
Sentinel Agent
哨兵		 平台政策、账户健康、风控异常无法实时预警。 老板、风控、财务、运营 “所有金融动作都先过风险边界。” Risk & Eligibility Monitor 持续监控异常, 让客户和银行更放心。
03 · L1-L4 Commercial Packaging

不同客户卖不同“包装”, 但底层都是 Agent OS

L1

AI Capital Desk

对象: 顶级品牌、Anker tier。卖点是资金效率、司库、风控和银行资金使用效率。

  • 入口: Capital Efficiency Assessment
  • 主推: Capital + Treasury + Sentinel
  • 成交: Enterprise subscription + custom dashboard
L2

Working Capital Map

对象: 工贸大卖。卖点是采购、供应商、库存、回款周期打通。

  • 入口: 现金周期诊断
  • 主推: Deal + Capital + Sentinel
  • 成交: Scale subscription + TermPay / MoreFund
L3

14-Day Growth Trial

对象: 成长品牌。卖点是增长和现金流同步。

  • 入口: 15 分钟增长资金诊断
  • 主推: Insight + Spend + Capital
  • 成交: Pro subscription + FastPay / MoreFund attach
L4

3-Min FastPay Starter

对象: 中小卖家。卖点是简单、快速、低门槛。

  • 入口: 3 分钟额度测算
  • 主推: Capital Lite + Sentinel Lite
  • 成交: Starter / Pro subscription + FastPay
