8TOP 5 风险 & 对策
致命
R1 · Amazon 官宣延期

L4 批量获客通道（GS-19/20）依赖 Amazon 时机。对策：非 Amazon 渠道（独立站卖家 / Shopify 生态）作为 Plan B，确保 L4 月新增 500 不依赖单一来源。

高
R2 · Agent 工具 W4 不可用

Insight Agent / Deal Agent 是销售提效核心。对策：W4 前用 Mock 版（规则引擎 + 人工评分）先行，W8 切换到真 Agent。

高
R3 · L1 周期长，12 周难闭单

L1 头部品牌平均销售周期 90-120 天。对策：12 周内"签 LOI + 启动 PoC"为主，正式合同延后；20 个 L1 目标只取 5 个全力闭。

中
R4 · 跨 sell 转化率不达标

Trio 跨 sell 是 ARR 倍增关键。对策：建立"客户成功 + AE"双触点，前 3 个月每月触发 1 次跨 sell 推荐。

中
R5 · AE 招聘速度跟不上

12 周需净增 6 名 AE / 4 名 SDR。对策：提前与 RPO 合作锁定 pipeline，并启用 Amazon 生态内推奖励。
