# Dowsure Growth Pack 访问说明

## 推荐入口

请优先打开:

`all_materials.html`

这是全部材料目录, 包含新版汇报、行动计划、市场方案、产品 Demo、CRM Demo 和旧版 01-08 材料。

## 本地打开

双击 `all_materials.html` 即可查看。

注意: 请保持整个 `growth_agent_outputs` 文件夹完整, 不要只单独拷贝一个 HTML 文件。页面之间使用相对链接, 样式文件是 `styles.css`。

## 发给别人下载查看

把整个 `growth_agent_outputs` 文件夹压缩成 zip 后发给别人。对方解压后双击:

`all_materials.html`

即可浏览全部内容。

## 在线共享推荐

最推荐方式:

1. 将整个 `growth_agent_outputs` 文件夹上传到静态网站托管服务。
2. 设置入口为 `all_materials.html`。
3. 把在线链接发给团队。

可选平台:

- 公司内网静态资源服务器
- 阿里云 OSS / 腾讯云 COS 静态网站
- Vercel / Netlify
- GitHub Pages
- 飞书知识库附件或网盘分享

## 说明

新版页面不依赖外部 JS 或后端服务, 下载后可以离线打开。部分旧版页面可能引用在线字体, 离线时字体可能回退, 但内容仍可查看。
